# Online Bouwmanager - Bouwkosten Calculator

Een moderne web applicatie voor het berekenen en beheren van bouwkosten voor verbouwingsprojecten.

## Features

- 📊 **Bouwkosten Calculator**: Bereken totale kosten voor uw bouwproject
- 📋 **Categorieën Management**: Voeg categorieën toe zoals materialen, arbeid, etc.
- 💰 **Kostenoverzicht**: Gedetailleerde breakdown van alle kosten
- 📈 **Budgetbewaking**: Houdt uw budget in de gaten
- 💾 **Opslaan & Delen**: Sla projecten op en deel deze met anderen

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript/React
- **Backend**: Node.js/Express (optioneel)
- **Database**: localStorage (client-side) of PostgreSQL (server-side)

## Installatie

```bash
git clone https://github.com/pietercleef-ux/Online-Bouwmanager.git
cd Online-Bouwmanager
npm install
npm start
```

## Gebruik

1. Maak een nieuw project aan
2. Voeg kostencategorieën in
3. Voeg uitgaven toe per categorie
4. Bekijk het totale overzicht
5. Sla uw project op
